
Aggiungi qui le vostre foto.

1) Copia i file .jpg/.png nella cartella.
2) Modifica `photos.json` affiancandoli così, ad esempio:
[
  { "src": "assets/photos/parigi-2023.jpg", "alt": "Noi a Parigi (2023)" },
  { "src": "assets/photos/mare-2020.jpg", "alt": "Tramonto al mare" }
]

Suggerimento: mantieni i nomi file senza spazi per semplicità.
